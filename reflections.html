Reflections

Before I signed up for this class, I had never thought that I can design a website by myself. Even though I'm always interested in Web design, I have never wanted to become a Web developer because I really hate coding. However, CIS241 class does give me some new thought about this field and what can I do in the future.

I have learnt a lot of interesting knowledge such as basic elements to create a website with HTML, how to format it using CSS, etc. I now also know that we don't have to know too much about coding to desgin a site because there are many useful tools out there that can be used to realese pressure for the desginers. Although I can now only create very simple and somehow boring website such as the one you're in right now, I'm much more confident that I can improve a lot hereafter if I continue studying

At the moment I construct this reflection page, I suddenly realize that we're going to the end of the quarter. The class is not so hard as I used to think before. I really enjoy the class and learn as much as I can. Though I may not choose to work as a Web developer, I believe that the knowledge I get from the class will support my career a lot in the future.
